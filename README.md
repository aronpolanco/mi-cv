# Interactive Digital CV - Aron Polanco 

Bienvenido al repositorio de mi currículum digital interactivo. Este proyecto fue desarrollado para mostrar mi perfil profesional, experiencia y habilidades técnicas de una manera moderna, limpia y accesible.

##  Live Demo
Puedes ver la versión en vivo de mi CV aquí: 
https://aronpolanco.github.io/mi-cv/

##  Tecnologías utilizadas
* **HTML5:** Estructura semántica y buenas prácticas.
* **CSS3:** Diseño responsivo, paleta de colores personalizada y animaciones (sin uso de frameworks externos).
* **JavaScript (Vanilla):** Lógica implementada para un sistema bilingüe (Español / Inglés) con transiciones suaves sin recargar la página.
